HTML Element Reference

Tag Name Description
<h1> html Defines an HTML document
<head> Head Contains metadata/information for the document
<title> Title Defines a title for the document
<body> Body Defines the document's body
<h1> to <h6> h1 to h6 Defines HTML headings
<p> p Defines a paragraph

CSS Properties

Property Value Description
background-color: Named color / RGB / HSL / HEX The background-color property specifies the background color of an element.
opacity: 0.0 - 1.0 / RGBA / HSLA The opacity property specifies the opacity/transparency of an element.
border-style: dotted / dashed / solid / double / groove / ridge / inset / outset / none The border-style property specifies what kind of border to display.
margin:, margin-top:, margin-right:, margin-bottom:,margin-left: auto / length / % / inherit The CSS margin properties are used to create space around elements, outside of any defined borders.
padding:,padding-top:,padding-right:,padding-bottom:,padding-left: length / % / inherit The CSS padding properties are used to generate space around an element's content, inside of any defined borders.
